* microutils

some micro tools written by LuHui <luhux76@gmail>


- ufbtest: framebuffer test

- umemdump: process memory dump

- memtest: memory test

- ustardict: dictionary

- utermhello: a curses-like hello world program, but not depend curses

** DEPEND

libc

** INSTALL

#+BEGIN_SRC shell
make install
#+END_SRC
